# Настройка AssemblyAI для транскрипции аудио

## Проблема
Если вы видите ошибку `500 Internal Server Error` при транскрипции, скорее всего не настроен API ключ AssemblyAI.

## Решение

### 1. Получите API ключ AssemblyAI
1. Зарегистрируйтесь на https://www.assemblyai.com
2. Перейдите в https://www.assemblyai.com/app/account
3. Скопируйте ваш API ключ

### 2. Настройте переменную окружения

#### Для локальной разработки:
Создайте файл `.env.local` в корне проекта:

```bash
ASSEMBLYAI_API_KEY=ваш-ключ-здесь
```

#### Для Vercel:
1. Перейдите в настройки проекта на Vercel
2. Environment Variables
3. Добавьте:
   - Name: `ASSEMBLYAI_API_KEY`
   - Value: ваш API ключ

### 3. Перезапустите сервер
После добавления `.env.local`:
```bash
# Остановите сервер (Ctrl+C)
# Запустите снова
npm run dev
```

## Проверка
После настройки в консоли сервера должно появиться:
```
✅ ASSEMBLYAI_API_KEY найден
```

Если ключ не найден, проверьте:
- Файл `.env.local` существует и находится в корне проекта
- Ключ указан правильно (без пробелов, кавычек)
- Сервер перезапущен после добавления ключа
