# Инструкция по запуску приложения

## Быстрый запуск

1. **Активируйте виртуальное окружение:**
   ```bash
   source venv/bin/activate
   ```

2. **Проверьте приложение перед запуском:**
   ```bash
   python check_app.py
   ```

3. **Запустите приложение:**
   ```bash
   streamlit run app.py
   ```

   Или используйте скрипт запуска:
   ```bash
   python run_app.py
   ```

## Если приложение не запускается

### 1. Проверьте зависимости
```bash
pip install -r requirements.txt
```

### 2. Проверьте Python версию
```bash
python --version
# Должна быть 3.8 или новее
```

### 3. Проверьте наличие всех файлов
- `app.py` - главный файл приложения
- `claude_assistant.py` - модуль ИИ
- `config.py` - файл конфигурации
- `modules/` - папка с модулями
- `utils/` - папка с утилитами

### 4. Проверьте ошибки
Запустите проверку:
```bash
python check_app.py
```

### 5. Запустите с выводом ошибок
```bash
streamlit run app.py --logger.level=debug
```

## Частые проблемы

### Проблема: "ModuleNotFoundError"
**Решение:** Установите недостающие модули:
```bash
pip install streamlit pandas numpy pillow plotly
```

### Проблема: "API key not found"
**Решение:** Проверьте файл `config.py` и наличие API ключей

### Проблема: "Port already in use"
**Решение:** Остановите другие процессы Streamlit или используйте другой порт:
```bash
streamlit run app.py --server.port 8502
```

## Альтернативный запуск

Если основной файл не работает, попробуйте упрощенную версию:
```bash
streamlit run app_simple.py
```
