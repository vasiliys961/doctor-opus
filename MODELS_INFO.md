# Информация о моделях ИИ

## Текущие модели в Next.js версии

### Для анализа изображений:
1. **Opus 4.5** (`anthropic/claude-opus-4.5`) — основной клинический ассистент
   - Используется для точного анализа
   - Поддерживает streaming
   - Максимальное качество диагностики

2. **Sonnet 4.5** (`anthropic/claude-sonnet-4.5`) — быстрый fallback
   - Используется как резервная модель
   - Быстрее Opus, но качество ниже

3. **Haiku 4.5** (`anthropic/claude-haiku-4.5`) — быстрый анализ документов
   - Для простых задач
   - Самый быстрый из Claude моделей

4. **Gemini Flash 2.5** (`google/gemini-2.5-flash`) — быстрый анализ
   - Используется для режима "быстрый анализ"
   - Не требует system_prompt
   - Компактные ответы

5. **Gemini Flash 3.0** (`google/gemini-3-flash-preview`) — новая версия
   - Улучшенная версия Flash
   - Используется по умолчанию для быстрого анализа

### Для ИИ-консультанта (чат):
- **Opus 4.5** — основной (по умолчанию)
- **Sonnet 4.5** — быстрый вариант
- Поддерживается streaming для постепенного получения ответа

## Режимы анализа

1. **Быстрый** (`fast`) — Gemini Flash
   - Быстрый скрининг
   - Компактное заключение
   - Для первичного просмотра

2. **Точный** (`precise`) — Opus 4.5
   - Детальный анализ
   - Максимальное качество
   - Для окончательного заключения

3. **С валидацией** (`validated`) — Gemini Flash + Opus
   - Сначала быстрый анализ через Gemini
   - Затем детальный анализ через Opus
   - Два заключения для сравнения

## Streaming

- Поддерживается для Opus и Sonnet
- Постепенное появление текста
- Помогает видеть ход рассуждений модели
