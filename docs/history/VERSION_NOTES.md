# Medical Assistant v3.30.0 - Claude 4.5 Update

### 1. Обновление моделей (Anthropic Claude 4.5)

  - *Результат*: Скорость выше, точность сопоставима, цена в 3 раза ниже.

Цены установлены согласно новым тарифам OpenRouter для Claude 4.5:

- **Haiku 4.5**: $0.40 / $2.00

- Обновлены лейблы в `AnalysisModeSelector.tsx`.
