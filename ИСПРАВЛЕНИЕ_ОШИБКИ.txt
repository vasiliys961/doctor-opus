ПРОБЛЕМА:
---------
Ошибка: "Cannot import 'setuptools.build_meta'"
Причина: Python 3.13 несовместим со старыми версиями numpy (1.24.3)

РЕШЕНИЕ:
--------

1. Установите setuptools и wheel:
   pip install setuptools wheel --upgrade

2. Установите пакеты по одному (без requirements.txt):
   pip install streamlit pandas numpy pillow plotly openpyxl python-docx pdfplumber PyPDF2 assemblyai audio-recorder-streamlit

3. ИЛИ просто запустите приложение - все уже работает!
   streamlit run app.py
